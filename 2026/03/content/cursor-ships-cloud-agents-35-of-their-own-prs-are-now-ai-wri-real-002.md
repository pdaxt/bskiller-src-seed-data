Cursor launched 'background agents' on Feb 24, 2026  -  cloud VMs that autonomously write code, run tests, and open PRs. The headline stat: 35% of Cursor's internal PRs now come from these agents. CNBC confirmed the launch. The agents run in cloud sandboxes with full dev environments. But here's the thing nobody's asking: 35% of PRs by count doesn't mean 35% of code value. A bot can open 100 PRs for lint fixes. The real question is what percentage of shipped features are agent-authored, and Cursor hasn't disclosed that.
